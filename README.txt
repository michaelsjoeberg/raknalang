# Raknalang (räknalang)

A simple arithmetic programming language aimed at learning basic mathematics in native language. Can be adapted to most languages.

## Examples (in Swedish)

	ett plus ett
	2

	två minus tre
	-1

	fem myror är fler än fyra elefanter
	Sant

	två lasbilar är färre än en bil
	Falskt

	två äpplen delat på två personer
	1 (kvar: 0)

	tre päron delat med två personer
	1 (kvar: 1)

	en glass delat med fyra personer
	0 (kvar: 1)
